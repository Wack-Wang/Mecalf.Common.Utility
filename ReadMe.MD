# 项目目的

## 为了方便开发使用，收集了常见的一些工具类，并进行了适当的修改。

# 包含的内容：

    目前包含了字符串的常见处理，如：判空、Json的序列化与反序列化、全角半角转换、长度计算（一个汉字占2个字节）；
    时间相关的有：Unix时间戳和DateTime的转换；
    安全相关的有：文本、数据流、文件的Md5计算；DES 加/解密，RSA 加/解密等。
    
    本项目在持续完善中，不定期更新。
